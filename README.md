# screen-saver

[![MIT license](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

> Starts the screensaver.

## Install

```bash
$ npm install --save screen-saver
```

## Usage

```js
const screenSaver = require('screen-saver');

// starts the screensaver
screenSaver();
```
